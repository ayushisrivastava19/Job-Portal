🚀 Job Portal
A full-stack Job Portal application built using React.js, Node.js, Express.js, MongoDB Atlas, Clerk Authentication, and Cloudinary. This platform allows job seekers to search and apply for jobs while recruiters can manage job postings and applications through a dedicated dashboard.

📦 Installation
Clone the Repository
```bash
git clone https://github.com/ayushisrivastava19/Job-Portal.git
```
Install Frontend Dependencies
```bash
cd Job-Portal/client
npm install
npm run dev
```
Install Backend Dependencies
```bash
cd ../server
npm install
npm run server
```
---
🔐 Environment Variables
Client (.env)
```env
VITE_CLERK_PUBLISHABLE_KEY=YOUR_KEY
VITE_BACKEND_URL=https://job-portal-ashy-three.vercel.app
```
Server (.env)
```env
MONGODB_URI=YOUR_MONGODB_URI
CLERK_SECRET_KEY=YOUR_SECRET_KEY
CLERK_WEBHOOK_SECRET=YOUR_WEBHOOK_SECRET
CLOUDINARY_NAME=YOUR_CLOUDINARY_NAME
CLOUDINARY_API_KEY=YOUR_API_KEY
CLOUDINARY_SECRET_KEY=YOUR_SECRET_KEY
JWT_SECRET=YOUR_JWT_SECRET
```
