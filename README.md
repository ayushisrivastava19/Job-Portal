# Job Portal

A responsive Job Portal frontend built using React and Tailwind CSS for browsing and managing job listings.

Designed as a beginner-to-intermediate level project for learning modern web development.

## Features
- Clean and responsive UI
- Job listings interface
- Modern design using Tailwind CSS

## Tech Stack
- React.js
- Tailwind CSS
- Vite

## Installation

git clone https://github.com/ayushisrivastava19/Job-Portal.git  
cd Job-Portal/client  
npm install  
npm run dev
