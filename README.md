# Job Portal

A full-stack Job Portal application built with React.js, Node.js, Express.js, MongoDB, Clerk Authentication, and Cloudinary.

## 🚀 Live Demo

### Frontend
https://job-portal-client-eta-nine.vercel.app

### Backend API
https://job-portal-ashy-three.vercel.app

## 📂 GitHub Repository

https://github.com/ayushisrivastava19/Job-Portal

## ✨ Features

### Job Seeker
- Browse available jobs
- Search jobs by title and location
- Apply for jobs
- Authentication using Clerk
- Responsive UI

### Recruiter
- Recruiter Login
- Add new jobs
- Manage posted jobs
- View applications
- Update application status

### General
- Secure Authentication
- Cloud Image Uploads with Cloudinary
- MongoDB Database Integration
- REST API Architecture
- Responsive Design

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/ayushisrivastava19/Job-Portal.git
```

### Frontend Setup

```bash
cd Job-Portal/client
npm install
npm run dev
```

### Backend Setup

```bash
cd Job-Portal/server
npm install
npm run server
```

---

## 🔐 Environment Variables

### Client (.env)

```env
VITE_CLERK_PUBLISHABLE_KEY=your_clerk_key
VITE_BACKEND_URL=your_backend_url
```

### Server (.env)

```env
MONGODB_URI=your_mongodb_uri
CLERK_SECRET_KEY=your_clerk_secret
CLOUDINARY_NAME=your_cloudinary_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_SECRET_KEY=your_secret_key
```
